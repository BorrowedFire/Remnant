# Code of Conduct

Remnant uses a simple maintainer-led conduct rule: be direct, respectful, and focused on the work.

Do not harass contributors, post private financial data, or use issues and pull requests to pressure maintainers into out-of-scope work. The maintainer may close, edit, hide, or lock discussions that violate these expectations.

For conduct concerns, open a minimal public issue asking for a maintainer contact path and do not include private details.
