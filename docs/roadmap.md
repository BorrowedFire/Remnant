# Remnant Roadmap

Remnant 1.0 is the open-source foundation for local books for solo businesses.

## 1.0 Foundation

- Review Inbox for imported expenses and cleanup issues
- CSV import with saved profiles
- Category presets with tax/reporting buckets
- Local accounts, vendors, clients, and projects as reporting dimensions
- Billable and reimbursable expense tracking
- Local document vault for receipts
- Apple Vision OCR for PDF/image receipts
- Basic `.eml` attachment import
- Backup and restore with integrity checks
- Raw expense CSV, tax-bucket summary CSV, and billable expense export
- OSS documentation, templates, and privacy-first contribution rules

## Deferred

- Gmail, Wave, bank, or hosted API connectors
- `.mbox` import and advanced email parsing
- Invoices
- Time tracking
- Income and cashflow
- Payroll
- Formal reconciliation
- Cloud sync or hosted collaboration
- Command palette and saved custom views

Large features require an accepted issue or RFC before implementation.
