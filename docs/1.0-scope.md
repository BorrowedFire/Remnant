# Remnant 1.0 Scope

Remnant 1.0 is a local-first macOS expense tracker for Borrowed Fire LLC. Its job is to replace Wave for expense review, receipt collection, and accountant-ready export, not to recreate Wave's full accounting suite.

## In Scope

- macOS app target only.
- Expense dashboard for local review status.
- Manual expense entry.
- Wave and bank CSV import into reviewed or draft expenses.
- Local receipt vault for downloaded receipt files.
- Receipt metadata extraction from text and PDF files.
- Receipt matching to expenses that are missing receipts.
- Single and batch draft-expense creation from downloaded receipt files.
- Bulk review actions for imported expenses.
- Local vendor rules for recurring merchant categorization.
- Tax-year CSV preview and export.
- Local-only data storage with no network calls, analytics, subscriptions, bank linking, or sync.

## Out Of Scope For 1.0

- Invoices, bills, payroll, income tracking, accounts receivable, accounts payable, profit and loss, and balance sheets.
- Gmail API, Wave API, bank API, or Computer Use automation.
- CloudKit, iCloud sync, hosted backend services, analytics, and subscriptions.
- OCR for scanned image receipts.
- Multi-user collaboration.
- Public App Store distribution.

## Release Gate

Before calling 1.0 ready, the app should pass:

- `xcodebuild -project Remnant.xcodeproj -scheme Remnant -configuration Debug -destination 'platform=macOS,arch=arm64' -derivedDataPath build/DerivedData test`
- `./script/build_and_run.sh --verify`
- A privacy/scope scan confirming the active app target has no network, cloud, subscription, bank-linking, iOS, bill, income, payroll, profit/loss, or invoice surface beyond documented negative privacy copy and parser stop words.
