# Security and Privacy Reporting

Remnant handles local financial records and receipt documents. Please do not include private financial data in public GitHub issues.

Report privately when a problem involves:

- Data loss or corrupted local records
- Accidental exposure of receipt, account, tax, or customer data
- Unexpected network, sync, analytics, or third-party data transfer
- Backup/restore integrity failures
- A vulnerability that should not be publicly disclosed before a fix exists

Email: admin@borrowedfire.com

For normal bugs, open a public issue with synthetic reproduction data.
